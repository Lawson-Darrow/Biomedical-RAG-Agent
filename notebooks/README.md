# Notebooks

Reproducible evaluation lives here. The eval notebook (Milestone 4-5) loads the
benchmark sets, runs the pipeline across frontier and open-weight models, and renders
the comparison tables that anchor the writeup.

- `eval.ipynb` — *(Milestone 4)* full metric suite on PubMedQA, one model.
- `comparison.ipynb` — *(Milestone 5)* frontier vs. open across the metric suite.
